# Mummadi Shiva Ganesh - AI Engineer Portfolio

This is a modern, animated portfolio built with Next.js, React, Tailwind CSS, and Framer Motion. It features a generative art hero section and a custom work experience timeline.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

## How to Push to GitHub

To push your changes to your GitHub repository, run the following commands in your terminal:

1. **Initialize Git (only if you haven't done this yet):**
   ```bash
   git init
   git remote add origin https://github.com/Mummadi-shiva-ganesh/your-repo-name.git
   ```

2. **Stage and Commit your changes:**
   ```bash
   git add .
   git commit -m "Update portfolio with new AI project details and 3D animations"
   ```

3. **Push to GitHub:**
   ```bash
   git branch -M main
   git push -u origin main
   ```

## Deployment

This project is configured for **Firebase App Hosting**. 

When you push your code to GitHub, if you have connected your repository to Firebase App Hosting in the [Firebase Console](https://console.firebase.google.com/), it will automatically build and deploy your site.

See `apphosting.yaml` for configuration details.
