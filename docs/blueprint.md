# **App Name**: Crescere Portfolio

## Core Features:

- Personal Information Display: Display personal information, including photo, 'About Me' section, and resume details.
- Project Showcase: Showcase projects with descriptions, tech stack, and links to live demos or repositories.
- Skills Visualization: Highlight technical skills and tools used.
- Internship Timeline: Detail internship experiences with responsibilities and achievements.
- Direct Contact Links: Implement a 'Connect' button that directs users to LinkedIn and GitHub profiles.
- Content Highlighter: Use a tool to intelligently extract keywords and phrases to highlight within the 'About Me' section and skills descriptions.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey professionalism and depth.
- Background color: Light gray (#F0F0F0), creating a clean, modern canvas.
- Accent color: Teal (#008080) to highlight key elements, offering a sense of calm and reliability.
- Body font: 'Inter' (sans-serif) for a clean, modern, and readable body text.
- Headline font: 'Space Grotesk' (sans-serif) for a computerized, techy feel.
- Use clean, minimalist icons to represent skills and technologies. 
- Implement a modern, grid-based layout to present information clearly and efficiently.
- Incorporate smooth, subtle animations and transitions using Barba.js to enhance user engagement and create a polished feel.
